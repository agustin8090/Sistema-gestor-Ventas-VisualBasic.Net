# Sistema-gestor-Ventas-VisualBasic.Net
Sistema completo de gestión de ventas desarrollado en Visual Basic .NET con base de datos SQL Server.
CRUD Completo: Productos, Clientes y Ventas

Gestión de Relaciones: Ventas por cliente y ventas por producto

Sistema de Búsqueda y Filtrado avanzado

Reportes y consultas personalizadas

 Stack Tecnológico
Lenguaje: Visual Basic .NET

Base de Datos: SQL Server

Arquitectura: Capas (Acceso a datos, Logica Negocio, Presentacion)

Características: Validaciones, consultas optimizadas

 Estructura de Datos
Productos: Catálogo con precios y stock

Clientes: Registro con datos completos

Ventas: Sistema transaccional con detalles de venta

Consultas: Relaciones cliente-venta y producto-venta
